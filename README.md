​المخزن الدائري هو مصفوفة ذات حجم ثابت نهايتها موصولة ببدايتها باستخدام معامل باقي القسمة.  
​خطوات الحل:
​إنشاء المخزن: يحتوي على مصفوفة للبيانات، مؤشر للكتابة، مؤشر للقراءة، وعداد للعناصر count لضبط السعة.  
​وتم تجهيز المخزن ليكون فارغاً تماماً في البداية
​ استقبال الاسم وإضافة اللاحقة "CE-ESY" له قبل التخزين.  
​
​إذا كان المخزن يتسع: يتم تخزين وقراءة الاسم كاملاً مع اللاحقة.  
​إذا كان المخزن لا يتسع: تظهر رسالة Buffer Overflow عند امتلاء السعة، ويتم تخزين جزء من النص فقط حسب الحجم المتاح.  
​وبعد إتمام عملية القراءة، يعود المخزن فارغاً تماماً (Underflow condition).
